# Estudo PHP com Banco de Dados / PHP Study with Database

Este repositório contém um projeto de estudo utilizando **PHP** e implementando um banco de dados. O objetivo principal é aprender como trabalhar com banco de dados em PHP, realizando operações como **CRUD** (Create, Read, Update, Delete) com um banco de dados relacional (MySQL).

### Funcionalidades
- Conexão com o banco de dados MySQL
- Operações CRUD básicas
- Uso de **PDO** (PHP Data Objects) para realizar a interação com o banco de dados
- Criação de formulários para inserção de dados
- Exibição de dados salvos no banco de dados

<hr>

This repository contains a study project using **PHP** and implementing a database. The main objective is to learn how to work with a database in PHP, performing operations such as **CRUD** (Create, Read, Update, Delete) with a relational database (MySQL).

### Features
- Connection to the MySQL database
- Basic CRUD operations
- Use of **PDO** (PHP Data Objects) to interact with the database
- Creation of forms for data entry
- Display of data saved in the database
